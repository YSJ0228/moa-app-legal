모아모아 — 데이터 삭제 요청

Moa Moa Personal Finance · 개발자: 모아모아 팀

한국어

모아모아는 계정을 유지한 채로도 데이터의 일부 또는 전체를 삭제할 수 있도록 다양한 옵션을 제공합니다.

1. 앱 내 직접 삭제 (즉시 처리)

지출 내역 삭제

  • 내역 탭 → 지출 행 길게 누르기 → 다중 선택 → 삭제
  • 또는 지출 카드 탭 → 우상단 메뉴 → 삭제

가계부 삭제

  • 설정 → 가계부 관리 → 길게 누르기 → 다중 선택 → 삭제
  • 또는 가계부 상세 → 위험 영역 → 가계부 삭제

영수증 이미지 삭제

  • 해당 지출 삭제 시 연결된 영수증 이미지도 자동 삭제

알림 삭제

  • 알림 인박스 → 선택 모드 → 삭제 (개별/일괄)

카테고리 삭제

  • 설정 → 카테고리 관리 → 개별 카테고리 삭제 또는 숨김 처리

AI 채팅 기록 삭제

  • 연결된 지출을 삭제하면 해당 채팅 메시지도 함께 삭제

2. 이메일 요청 (특정 데이터 일괄 삭제)

앱 내에서 처리하기 어려운 데이터 삭제는 이메일로 요청해주세요:

  • 받는 사람: moa.official.kr@gmail.com
  • 제목: [모아모아] 데이터 삭제 요청
  • 본문에 다음 내용을 포함해주세요:
    • 가입 이메일 주소
    • 삭제하고 싶은 데이터 (예: 2025년 이전 모든 지출, 특정 가계부의 영수증 전부 등)
📩 요청 접수 후 7일 이내 처리됩니다.

3. 보관되는 데이터

다음 데이터는 관련 법령에 따라 일정 기간 보관 후 자동 폐기됩니다:

  • 전자상거래 결제·환불 기록: 5년 (전자상거래법 제6조)
  • 접속 로그·IP: 3개월 (통신비밀보호법 제15조의2)

위 데이터의 개인정보(이름·이메일)는 익명화 처리됩니다.

4. 전체 계정 삭제를 원하신다면

모든 데이터를 한 번에 삭제하시려면 계정 삭제를 진행하세요: → 계정 삭제 요청 페이지

English

Moa Moa allows you to delete part or all of your data while keeping your account active.

1. In-app deletion (instant)

Delete expense entries

  • History tab → long-press an expense → multi-select → Delete
  • Or tap an expense card → top-right menu → Delete

Delete a book (account-book)

  • Settings → Manage books → long-press → multi-select → Delete
  • Or open book detail → Danger zone → Delete book

Delete receipt images

  • Deleting an expense also removes its attached receipt image

Delete notifications

  • Notification inbox → selection mode → Delete (individual or bulk)

Delete categories

  • Settings → Manage categories → delete or hide individual categories

Delete AI chat history

  • Deleting the linked expense also removes the related chat message

2. Email request (bulk-specific data)

For data you cannot delete in-app, email us:

  • To: moa.official.kr@gmail.com
  • Subject: [Moa Moa] Data deletion request
  • Please include:
    • Email used to sign up
    • What data to delete (e.g., all expenses before 2025, all receipts in a specific book, etc.)
📩 Requests are processed within 7 days.

3. Data that will be retained

The following are retained for a limited period as required by Korean law, then permanently deleted:

  • E-commerce payment/refund records: 5 years (Act on Consumer Protection in Electronic Commerce, Article 6)
  • Access logs / IP: 3 months (Protection of Communications Secrets Act, Article 15-2)

Personal identifiers (name, email) in the retained data are anonymized.

4. Want to delete the whole account?

To delete everything at once, see our account deletion page: → Account deletion request
